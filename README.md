# pcsx2-texture-swapper-cpu
Ferramenta em Python para automatizar a remasterização de texturas do emulador PCSX2. Utiliza DreamShaper 8 via OpenVINO para geração local em CPU e BLIP para legendar e entender o contexto das texturas originais (Image-to-Image). Inclui modos de geração total, parcial e "sem contexto" para resultados variados e estilizados.
